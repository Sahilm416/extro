# @extro/types

Shared TypeScript types for [Extro](https://github.com/Sahilm416/extro). Contains `ManifestV3` and `ExtroConfig` shapes. Pure types, no runtime.

This package is an internal dependency of `extro`, `@extro/vite-plugin`, and `@extro/react`. You usually don't need to install it directly.
